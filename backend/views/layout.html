ApostropheCMS Backend for Astro

You are currently viewing the ApostropheCMS backend, used to manage the content for your project. It supplies the Admin UI and in-context editing functionality. The frontend rendering of the website is handled by Astro and is delivered from the Astro project in the "frontend/" folder.

Important Note About Frontend Server

This status checker only works when the Astro frontend is running in development mode (npm run dev). It will not detect servers running in preview mode (npm run preview) or production mode (npm run serve).

For local development and content editing, please ensure you're using npm run dev to start the Astro frontend.

Setting up the Astro Frontend Project

To view the complete project, you need to run the Astro frontend. Follow these steps:

  1. Navigate to the frontend directory: cd frontend
  2. Install dependencies: npm install if not already done
  3. Start the Astro development server: npm run dev

The Astro frontend is running. You can access the complete site (default configuration) at: http://localhost:4321
